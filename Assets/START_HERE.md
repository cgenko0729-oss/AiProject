# START_HERE.md

## How to Use These Files

Put these files in your Unity project root:

```text
SPEC.md
SKILL.md
CLAUDE.md
PROMPT_FOR_CLAUDE_CODE.md
START_HERE.md
```

Then open Claude Code in the Unity project root.

Start with the Phase 0 prompt in:

```text
PROMPT_FOR_CLAUDE_CODE.md
```

## Important

Do not ask Claude:

```text
Finish the whole project.
```

Instead, do it phase by phase:

```text
Phase 0: Read and plan only
Phase 1: Folder and asmdef
Phase 2: Simple Core skeleton
Phase 3: Tests first
Phase 4: Core implementation
Phase 5: Simple 3D presentation
Phase 6: Constraint check and simplify
Phase 7: Final review
```

## Project Direction

This version assumes:

- 3D dungeon visual style
- loose reference: 風来之国 dungeon battle feeling
- fixed-seed turn-based grid roguelike logic
- KISS principle
- few classes
- simple English names
- simple Japanese comments
